# JobFinder

基于 Selenium + ChatGPT 的职位搜索与简历生成工具。自动从 Seek 抓取职位、调用 ChatGPT 生成定制简历与求职信，并导出至 Excel 与 PDF。

---

## 环境要求

- **Python 3.8+**
- **Google Chrome**（需已安装）
- **ChatGPT 账号**（需已登录 chat.openai.com 或 chatgpt.com）

## 安装步骤

### 1. 克隆或下载项目

```bash
git clone <repository-url>
cd JobFinder
```

### 2. 创建虚拟环境（推荐）

```bash
python -m venv venv
venv\Scripts\activate   # Windows
# source venv/bin/activate   # macOS / Linux
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

依赖包括：`openpyxl`、`selenium`、`webdriver-manager`。

### 4. 准备 skill.md（可选）

在项目根目录创建 `skill.md`，写入你的技能、经历摘要。ChatGPT 会据此生成简历，减少虚构内容。

## 配置

### 方式一：通过 UI 配置（推荐）

1. 运行程序：`python jobfinder_ui.py` 或双击 `jobfinder_launcher.pyw`
2. 在界面中填写：
   - **Basic**：职位地点、关键词、运行次数
   - **Advanced**：Excel 路径、Chrome 路径、ChatGPT 对话名称等
   - **Personal Info**：姓名、电话、邮箱、地址（用于简历 Header）
3. 点击「保存配置」

### 方式二：编辑 config.json

可参考 `config.example.json`，主要字段：

| 字段 | 说明 |
|------|------|
| `output_excel` | 输出 Excel 路径 |
| `job_location` | 职位地点（如 "All Gold Coast QLD"） |
| `keyword` | 搜索关键词 |
| `max_runs` | 最大处理职位数（留空=不限） |
| `single_job_url` | 单条 JD URL 模式，只处理该职位 |
| `skip_title_contains` | 跳过标题包含的关键词（逗号分隔） |
| `chrome_path` | Chrome 可执行文件路径 |
| `chatgpt_chat_title` | ChatGPT 对话名称（留空=使用当前对话） |

## 运行流程

### 1. 启动 Debug Chrome

在 UI 中点击「启动 Debug Chrome」，会打开带调试端口的 Chrome 实例。

### 2. 登录 ChatGPT

在打开的 Chrome 中访问 chat.openai.com 或 chatgpt.com，完成登录。

### 3. 开始运行

点击「开始运行」。程序会：

1. 在 Seek 上应用搜索条件
2. 逐页抓取职位链接
3. 逐个打开 JD，提取信息
4. 将 prompt 发送给 ChatGPT
5. 解析返回的 JSON，写入 Excel
6. 可选导出简历与求职信 PDF

### 4. 单条 URL 模式

在「运行次数」输入框填入单个 JD URL（如 `https://www.seek.com.au/job/12345678`），程序会只处理该职位。

## 输出文件

- **Excel**：`job_results.xlsx`（或 config 中配置的路径）
- **PDF**：`pdf_output/<job_id>/` 目录下的简历与求职信

## 常见问题

### CMD 窗口弹出

- 使用 `pythonw jobfinder_ui.py` 或双击 `jobfinder_launcher.pyw` 启动，可避免控制台窗口。

### ChatGPT 回复慢

- 将「ChatGPT 对话名称」留空，运行前手动新建一个对话，程序会使用当前对话，避免长对话变慢。

### 找不到 Chrome

- 在 Advanced 中填写 Chrome 路径，常见默认：`C:\Program Files\Google\Chrome\Application\chrome.exe`
